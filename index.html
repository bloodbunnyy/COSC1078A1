Project Ideas




For my project idea, I am interested in using the raspberry pi to create a hidden camera device that streams the footage directly to your mobile phone. This will involve programming the raspberry pi using java language, attaching a camera to it and once getting that to work, figuring out how to get it to send the video stream to a mobile phone. I would also like to be able to have it set up in case/box so that the camera can actually be hidden and out of plain site.

I believe this product would be useful in many ways. The main reason which sparked my idea for this product in the first place, is to be able to check up on your pets while you’re not at home. Whenever I’m away from home I’m always worried, thinking about if my children are okay and if they’re staying out of trouble. With this project, I’d be able to check up on them at any time with my mobile device even when I’m not at home, leaving me feeling at ease.

The product will come already installed, enclosed within a box that the end user will be able to put anywhere they like within their home. The user will then download an application on their mobile phone and connect to the device via a cable for initial setup. Once the setup is complete, the device will be connected to the home wireless internet network and be ready for use. The user will then be able to go into their application in order to activate the device and view what the device is currently recording. The device will also intermittently take photos throughout the day that the user can check on later in their own time. The photos will be stored locally on a USB drive connected to the raspberry pi (1 will be shipped with the device). The user will be able to activate motion detection if they wish, which will be recording if there is motion detected. The app will allow the user to access multiple cameras in their home and will also have access to some functionality that will allow them to zoom in and out. In order to deliver these services to the app, the device will run on a server that connects to the internet to allow the user to get a video feed with very minimal delay wherever they may be in the world (if they have some sort of internet connection). The device will have 2 primary cameras, one being a regular camera and the other one, an infrared camera. This will allow the device to function even if it is in a dark place i.e. it is late at night or the camera is just stationed in a dark room or space such as inside a safe. The smart device application will feature a simple interface that allows the user to easily switch between cameras and view past photos or videos. It will also have the ability to share photos and videos taken onto the user’s social media accounts if they wish. The design of the box/case that it comes in will be subtle as to fit in with other things the user might have stored near it. It will also be designed in a way to allow as wide a view as possible as to limit the amount of dead space in the area.

To create my project, I will first need to purchase a raspberry pi. Since I haven’t used one before I will have to do my research and watch videos to learn how they work and how to operate them. I will need to purchase a camera of some sorts, preferably a something small so that I can make it a hidden camera but if I can’t find anything similar, I could use a web cam. I already know some java basics but would have to study up to be able to write program for my idea to function. I would also have to figure out how to be able to get the feed from the camera to be sent directly to my phone. I would have to investigate this, maybe there is an app I can already use, or I may have to create my own, or maybe I could just get it to work by the camera taking short recordings and sending them via cloud to my phone. I don’t think it will be a very hard project to achieve as I know the raspberry pi already has a slot to attach a camera, it’s just about figuring out how everything else will come together to bring my idea to life.

If the project is successful, you should be able to set it up anywhere in your home and connect it up to your Wifi. Once turned on, the device should then be able to live record its surroundings, sending a direct feed to your mobile device. I think this project will be very useful to me and others as purchasing surveillance cameras that can be viewed via your phone currently are quite expensive. By creating my own using the raspberry pi, it will not only be a fun project, it will be a lot more affordable. With this device you’ll be able to check up on your pets, see when your food is finished cooking from another room, be able to check up on carers/babysitters to make sure they’re doing their job correctly or it can even be placed at your front door so you can see if there is any suspicious activity happening around your home when you’re not there.
